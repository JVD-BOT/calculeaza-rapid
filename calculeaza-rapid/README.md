# CalculeazăRapid — Instrumente Financiare România 2026

Romanian financial tools hub with salary calculator, PFA tax estimator, and mortgage simulator.

## Features

- **Calculator Salariu** — Brut ↔ Net conversion (2026 tax code: CAS 25%, CASS 10%, impozit 10%)
- **Calculator PFA** — Freelancer tax estimator (Sistem Real & Normă de Venit)
- **Simulator Credit Ipotecar** — Mortgage calculator with visual breakdowns
- SEO-optimized for Romanian search queries
- Mobile-first responsive design
- Zero backend — pure frontend, free hosting

## Tech Stack

- React 19 + Vite
- Google Fonts (Instrument Serif + DM Mono)
- Pure CSS-in-JS (no dependencies)
- Deployed on Vercel (free tier)

## 🚀 Deploy for FREE — Step by Step

### Option A: Deploy to Vercel (Recommended)

1. **Create a GitHub account** (if you don't have one): https://github.com/signup

2. **Install Git** (if needed):
   - Windows: Download from https://git-scm.com/download/win
   - Mac: `brew install git` or download from https://git-scm.com/download/mac

3. **Push this project to GitHub:**
   ```bash
   cd calculeaza-rapid
   git init
   git add .
   git commit -m "Initial commit - CalculeazăRapid"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/calculeaza-rapid.git
   git push -u origin main
   ```

4. **Deploy on Vercel:**
   - Go to https://vercel.com and sign up with your GitHub account
   - Click "Add New → Project"
   - Import your `calculeaza-rapid` repository
   - Vercel auto-detects Vite — just click "Deploy"
   - Your site is live at `calculeaza-rapid.vercel.app` in ~60 seconds!

5. **Add your custom domain (optional, ~€8/year):**
   - Buy `calculeazarapid.ro` from a registrar (e.g., rotld.ro, namecheap.com)
   - In Vercel dashboard → Settings → Domains → Add `calculeazarapid.ro`
   - Update DNS as instructed by Vercel

### Option B: Deploy to Netlify

1. Same GitHub steps as above
2. Go to https://app.netlify.com
3. Click "Add new site → Import an existing project"
4. Connect GitHub and select your repo
5. Build command: `npm run build`
6. Publish directory: `dist`
7. Click "Deploy site"
